CKLabs API Reference

This is a web scraping tool, written in Django, that constantly scrapes the articles of a major technology blogs and stores it in a PostgreSQL database.

These are the relevant data being stored:

  • Outlet name and metadata (Title, website and description);
  • Authors (name, outlet, profile page, about, avatar and social networks);
  • Published articles and metadata (title, publication date, author, content, thumbnail, url, outlet, authors, categories);
  • Categories' tags

Currently we're scraping content from:

  • TechCrunch
  • Cheesecake Labs
  • Mashable
  • Engadget

This rest API, provides access to all these informations.

API Endpoint
http://cheesecake-challenge.herokuapp.com/
